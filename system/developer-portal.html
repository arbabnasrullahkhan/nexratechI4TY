Authentication

All API requests must include an Authorization header utilizing a Bearer token issued via Firebase Authentication.

HTTP Request 
GET /api/v1/protected-resource HTTP/1.1
Host: api.nexratech.pk
Authorization: Bearer YOUR_JWT_TOKEN

Fetch Products

Retrieve the master product catalog. Supports pagination and category filtering.

GET /api/v1/products 
{
  "status": 200,
  "data": [
    {
      "id": "prod_xyz",
      "title": "SaaS Starter Kit",
      "price": 25000,
      "stock": 50
    }
  ]
}

Order Submission

Create a new order securely. Requires a valid cart payload and signed payment intent.

POST /api/v1/orders 
{
  "userId": "uid_12345",
  "items": [
    { "productId": "prod_xyz", "qty": 1 }
  ],
  "paymentIntent": "pi_3MtwBw..."
}

Webhooks

Listen for asynchronous events such as order fulfillment and license generation.

order.completed
Fired when a payment is successful and assets are delivered.
license.revoked
Fired when an admin revokes a software key.
